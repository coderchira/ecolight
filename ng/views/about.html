Ecolight

This is a small scale household IoT project, under categorised energy optimization & digitization. To better describe it - Cloud connected LED bulb driven by single board computer striving to optimize the "on" time against user driven preferences and the ambient threshold. Cloud connected setup would not only let the user set the preferences remotely , but also monitor the room conditions on real time. Ecolight serves wide range of photo-ambiance , with easy access to the calibration and configuration of the setup..
